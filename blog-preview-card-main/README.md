# Cartão de Pré-visualização do Blog

Este é um projeto de **cartão de pré-visualização de blog**, desenvolvido com **HTML5 e CSS3**, inspirado em um desafio da plataforma **Frontend Mentor**.  
O objetivo é praticar **estrutura semântica, estilização com CSS e layout responsivo**.

---

## Visão Geral

O projeto exibe um card contendo:
- Imagem ilustrativa do artigo
- Categoria destacada
- Data de publicação
- Título do artigo com efeito hover
- Informações do autor (avatar e nome)

## Tecnologias Utilizadas

- **HTML5**
- **CSS3**
- **Flexbox**
- **Google Fonts (Figtree)**
- **Variáveis CSS**
- **Hover Effects**

## Objetivos do Projeto

- Praticar **HTML semântico**
- Trabalhar com **centralização usando Flexbox**
- Utilizar **variáveis CSS** para organização de cores
- Criar um layout limpo e fiel ao design proposto
- Melhorar a estrutura e legibilidade do código
